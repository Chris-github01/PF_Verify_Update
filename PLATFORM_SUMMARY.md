## 🔄 THE 8-STEP WORKFLOW

```
1. IMPORT → Upload PDFs/Excel quotes (AI extracts everything)
2. CLEAN → Normalize units, fix typos, group items
3. VALIDATE → Detect systems, verify compliance
4. ANALYZE → AI flags gaps, risks, over-pricing
5. MATRIX → Line-by-line comparison across suppliers
6. SCORE → 0-100 risk score per supplier
7. REPORT → Board-ready award recommendation (PDF)
8. EXPORT → Excel, PDF, Word outputs
```

**Time:** 30 minutes | **Manual:** 2-3 weeks
