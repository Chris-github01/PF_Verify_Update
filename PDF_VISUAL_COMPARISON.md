### AFTER (Unified)

- Consistent header on every page
- Proper footer with supplier name
- Single "Page X of Y" counter
- No redundant branding in content

## 7. Empty Page Handling

### BEFORE
```
Page 8:
[Section with 3 items]

Page 9:
[COMPLETELY EMPTY - only header/footer]

Page 10:
[Next section starts]
```
**Issue**: Wasted paper, unprofessional appearance

### AFTER
```
Page 8:
[Section with 3 items]

[Next section merged onto same page if <25% fill would occur]

Page 9:
[Next section starts - no empty pages]
```
**Improvements**:
- Smart merging prevents orphan pages
- More compact PDFs
- Professional appearance

## 8. Typography Scale

### Senior Theme Typography
```
Cover Title: 46pt, bold
Section Title: 15pt, bold
Table Header: 10pt, bold, uppercase
Table Body: 9pt, regular
Caption/Badge: 8pt, medium
```

### Junior Theme Typography
```
Cover Title: 46pt, bold (same)
Section Title: 16pt, bold
Table Header: 11pt, bold, uppercase
Table Body: 10pt, regular
Caption/Badge: 9pt, medium
```

**Reasoning**:
- Senior: Maximum information density for desktop review
- Junior: Enhanced readability for on-site/tablet use

## Summary of Visual Changes

| Element | Before | After |
|---------|--------|-------|
| Row Icons | ✓ in every row | Badge at section level |
| Header/Footer | Inconsistent | Unified "Page X of Y" |
| Table Headers | Split across pages | Repeat on every page |
| Pagination | Random splits | Smart min-6-rows rule |
| Empty Pages | Common | Prevented via merging |
| Typography | One-size-fits-all | Theme-based sizing |
| Section Spacing | Inconsistent | Theme-controlled |
| Visual Hierarchy | Weak | Strong with badges |
