## Typography Scale

### Headings
- **H1 (Page Title):** 42px, 800 weight, -1px letter spacing
- **H2 (Section):** 28px, 700 weight, -0.5px letter spacing
- **H3 (Subsection):** 20px, 600 weight
- **H4 (Detail):** 16px, 600 weight

### Body
- **Large:** 15px, 400 weight, 1.7 line height
- **Regular:** 14px, 400 weight, 1.6 line height
- **Small:** 13px, 400 weight, 1.6 line height
- **Tiny:** 11-12px, 500 weight, uppercase, 0.5px letter spacing

### Special
- **Stat Numbers:** 32px, 800 weight, orange
- **Card Values:** 20px, 700 weight
- **Table Text:** 13px, 400 weight

---

## Print Settings Recommendation

When printing to PDF, recommend these settings:

- **Paper Size:** A4 (or Letter)
- **Margins:** Default (15-20mm)
- **Scale:** 100%
- **Background Graphics:** ON (important for colors!)
- **Headers/Footers:** OFF (we provide our own)
- **Color Mode:** Color (not grayscale)
- **Quality:** High
